## picgo-plugin-elegant-rename

A PicGo plugin for elegant name.



针对picgo写的插件，用于规范图床名称与[md-writer-qiniu](https://github.com/chenghm123/md-writer-qiniu)一致。

以后不用Atom写markdown，强烈推荐Typora。



---

|     参数     |     默认值     | 说明                        |
| :----------: | :------------: | --------------------------- |
| prefix | '' | 文件名个性前缀格式 |
